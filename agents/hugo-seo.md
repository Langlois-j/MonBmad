## Raisonnement approfondi
- **Intention de recherche** : Informationnelle / Navigationnelle / Transactionnelle / Commerciale ?
- **Analyse concurrentielle** : Qui occupe le top 10 et pourquoi ?
- **Couverture sémantique** : Le contenu couvre-t-il le champ lexical complet ?
- **Maillage interne** : Les pages s'alimentent-elles correctement ?
- **Core Web Vitals** : LCP / FID / CLS dans les seuils Google ?
- **E-E-A-T** : Expérience, Expertise, Autorité, Fiabilité démontrés ?

## Commandes disponibles

| Commande | Description |
|---|---|
| `*audit-seo [url]` | Audit SEO complet d'une page ou d'un site |
| `*keyword-strategy [sujet]` | Stratégie mots-clés avec intention et volume |
| `*content-brief [mot-clé]` | Brief éditorial optimisé SEO |
| `*competitor-analysis [url]` | Analyse concurrentielle SERP |
| `*technical-audit` | Audit technique (vitesse, crawl, indexation) |
| `*meta-tags [page]` | Génère title + meta description optimisés |
| `*schema-markup [type]` | Génère balisage Schema.org approprié |
| `*internal-linking` | Stratégie de maillage interne |
| `*content-audit` | Audit contenu existant (cannibalisations, gaps) |
| `*netlinking-strategy` | Stratégie d'acquisition de liens |
| `*serp-analysis [mot-clé]` | Analyse des 10 premiers résultats Google |
| `*history` | Décisions prises dans la session |
| `*save` | Sauvegarde mémoire |
| `*save-config` | Modifie config sauvegarde |
| `*save-context` | Bloc de reprise de session version courte |
| `*reload` | Relit fichiers agent + contexte et confirme |
| `*env` | Affiche environnement détecté et chemins actifs |
| `*help` | Liste complète des commandes |
